Teachable Machine Project

A no-code approach to build ML models

What is Teachable Machine?

Teachable Machine is a web-based tool by Google that allows anyone to create simple machine learning models without writing a single line of code. It supports training models for image classification, sound recognition, and pose detection directly in your browser, using your webcam or uploaded data.

Under the hood, it uses TensorFlow.js to handle the training and testing of models live on your device. This means your data stays private and models can be tested instantly, providing a smooth learning experience for beginners to understand AI concepts like data bias, overfitting, and classification pipelines.

How Teachable Machine Works

  • Collect labeled data via webcam or file uploads
  • Use transfer learning to fine-tune a pre-trained neural network in the browser
  • Test results instantly with live predictions and confidence scores
  • Export your trained model to web (TensorFlow.js), mobile (TensorFlow Lite), or even CoreML for iOS

How to Build Your Own

  1. Go to Teachable Machine
  2. Choose your project type: Image, Audio, or Pose
  3. Capture or upload data for at least 2 classes
  4. Train your model until accuracy is acceptable
  5. Test and improve using live predictions
  6. Export your model to integrate with your projects

Benefits & Best Practices

  • Perfect for beginners and non-coders
  • Data is handled privately in your browser
  • Export to multiple platforms: web, Android, iOS
  • Collect balanced datasets to avoid bias
  • Use at least 30–50 samples per class for robust results
  • Explore data augmentation techniques to improve performance

Using Your Exported Model

If you want to use your exported TensorFlow.js model in your own webpage, here’s a minimal code snippet:


const modelURL = "model.json";
const metadataURL = "metadata.json";
const model = await tmImage.load(modelURL, metadataURL);
const prediction = await model.predict(webcamElement);
